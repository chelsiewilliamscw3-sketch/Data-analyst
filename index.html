Quarterly Variance SQL Report

SQL-based quarterly variance calculations used to support financial dashboards and executive reporting.

SQL CTEs Finance Analytics
  • Created quarterly roll-ups with revenue, COGS, and OpEx
  • Calculated Net Income variance by quarter
  • Exported results to Excel & Tableau for reporting
See SQL Snippets

SQL Snippets

WITH m AS (
  SELECT YEAR([Date]) AS [Year], DATEPART(QUARTER, [Date]) AS [Quarter], Revenue, COGS, OpEx
  FROM dbo.Finance
)
SELECT [Year], [Quarter], SUM(Revenue) AS Revenue,
       SUM(Revenue-COGS-OpEx) AS NetIncome
FROM m
GROUP BY [Year], [Quarter]
ORDER BY [Year], [Quarter];
